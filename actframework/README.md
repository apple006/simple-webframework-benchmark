# [ActFramework](http://actframework.org) Test Bed

## Listening port

ActFramework application by default listen on port `5460`

## Run the app in PROD mode

### build and run by yourself

```
mvn clean package
cd target/dist
unzip *
./start
```

### use the `run.sh` script

```
./run.sh
```
